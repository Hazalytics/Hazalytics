# Hazalytics

> **Find the Edge**

Hazalytics is a sports analytics toolkit built in Python, designed to identify value through data, probability and mathematical modelling.

## 🚀 Features

- Odds Converter
- Probability Calculator
- Expected Value (EV) Calculator
- Fair Odds Calculator
- Kelly Criterion Calculator
- 📊 Bet Analyser
    - Implied Probability
    - Model Probability
    - Edge %
    - Fair Odds
    - Expected Value
    - Kelly Staking
    - Recommended Stake
    - Risk Rating
    - Confidence Score
    - Value Grade
    - Verdict
